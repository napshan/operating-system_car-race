遊戲需要的音效文件清單
====================

請將以下音效文件放在與 main.exe 相同的目錄中：

【必需文件】（如果沒有，遊戲會使用 Beep 備用音效）

1. engine_idle.wav 或 engine_idle.mp3
   - 引擎怠速/運轉音效
   - 用途：車子移動時持續播放
   - 建議：循環播放的引擎運轉聲
   - 長度：建議 2-5 秒（會自動循環）

2. engine_accel.wav 或 engine_accel.mp3
   - 引擎加速音效
   - 用途：按加速鍵時播放（與運轉聲同時播放）
   - 建議：循環播放的引擎加速/轉速上升聲
   - 長度：建議 2-5 秒（會自動循環）

3. brake.wav 或 brake.mp3
   - 煞車音效
   - 用途：按左右鍵（A/D 或 ←/→）轉向時播放
   - 建議：短促的煞車聲
   - 長度：建議 0.3-1 秒

4. bgm.mp3 或 bgm.wav
   - 背景音樂
   - 用途：遊戲開始後自動循環播放
   - 建議：適合賽車遊戲的背景音樂
   - 長度：任意（會自動循環）

【可選文件】（如果沒有，會使用 Beep 備用音效）

5. crash.wav 或 crash.mp3
   - 碰撞音效
   - 用途：撞到障礙物或邊界時播放
   - 建議：短促的碰撞聲（0.5-2 秒）

6. gameover.wav 或 gameover.mp3
   - GAME OVER 音效
   - 用途：遊戲結束時播放
   - 建議：戲劇性的失敗音效（1-3 秒）

7. victory.wav 或 victory.mp3
   - 勝利音效
   - 用途：完成賽道時播放
   - 建議：勝利音樂或音效（2-5 秒）

文件格式建議：
-------------
- WAV 格式：兼容性最好，推薦用於音效（engine.wav, crash.wav 等）
- MP3 格式：文件較小，推薦用於背景音樂（bgm.mp3）
- 採樣率：建議 44100 Hz
- 位深度：16-bit
- 聲道：單聲道或立體聲均可

文件放置位置：
-------------
所有音效文件應該放在與 main.exe 相同的目錄中，例如：

C:\Users\15507\Downloads\os_hw\
├── main.exe
├── engine.wav      ← 放在這裡
├── bgm.mp3         ← 放在這裡
├── crash.wav       ← 放在這裡（可選）
├── gameover.wav    ← 放在這裡（可選）
└── victory.wav     ← 放在這裡（可選）

最小配置：
---------
如果只想放最少的文件，至少需要：
- engine_idle.wav（引擎運轉聲）
- engine_accel.wav（引擎加速聲，可選但建議）
- bgm.mp3（背景音樂）

其他音效會使用 Beep 備用方案。

測試方法：
---------
1. 將音效文件放在正確位置
2. 運行遊戲
3. 如果聽到自定義音效，說明文件加載成功
4. 如果聽到 Beep 聲，說明文件未找到或格式不支持
